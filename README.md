# Drone route calculator
Github for the Werktuigkunde en Elektrotechniek drone project 2026
(Currently work in progress.)
## Requirements
To run the script Python 3 will be needed, as well as the two following libraries: [rich](https://github.com/textualize/rich) and [pyqt6](https://pypi.org/project/PyQt6/)


## Usage
To test it, just run algo_ui.py 

If you like command line, you can run the CLI version (algo.py) though you will have to change the variables manually in the files.
